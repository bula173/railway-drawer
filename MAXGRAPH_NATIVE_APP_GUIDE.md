# Railway Drawer: Native maxGraph Application Guide

## Overview

This is a **complete rewrite** of Railway Drawer using **ONLY maxGraph's native components**. No custom implementations, no manual UI building, no boilerplate code.

**Philosophy:** Configuration over Code

## Architecture

```
┌─────────────────────────────────────┐
│  RailwayDrawerMaxGraphApp           │ React Component
├─────────────────────────────────────┤
│  maxGraph Editor (configured)       │ Main Orchestrator
├──────────────┬──────────────┬───────┤
│  Toolbar     │ Format Panel │Outline│ Auto-generated from config
│(auto-gen)    │ (auto-gen)   │(auto) │
├──────────────────────────────────────┤
│  Graph (Drawing Engine)              │ Canvas
├──────────────────────────────────────┤
│  Status Bar (React state)            │ Live Updates
└──────────────────────────────────────┘
     ↑
     │ Configuration
     │
 railwayDrawerConfig.xml
```

## Key Components

### 1. RailwayDrawerMaxGraphApp.tsx (300 lines)

**Pure maxGraph Architecture:**

```typescript
// Load configuration
const configXml = fetch('/config/railwayDrawerConfig.xml');

// Create editor with config - that's IT!
const editor = new Editor(configXml);

// maxGraph handles:
// ✅ Toolbar generation
// ✅ Format panel creation
// ✅ Outline view
// ✅ Keyboard shortcuts
// ✅ Undo/redo
// ✅ Copy/paste
// ✅ Selection
// ✅ All actions
```

**What this component does:**
1. Loads XML config
2. Creates Editor instance
3. Mounts containers
4. Listens to events for React state updates
5. Provides save/load/export handlers

**What it DOESN'T do:**
- ❌ Build toolbar manually
- ❌ Handle keyboard events
- ❌ Manage undo/redo
- ❌ Create format panels
- ❌ Manage selection
- ❌ Define shortcuts

### 2. railwayDrawerConfig.xml (300 lines)

**Single source of truth for all features:**

#### Keyboard Shortcuts (Auto-wired)
```xml
<keyhandler>
  <add as="Ctrl+Z" action="undo"/>
  <add as="Ctrl+C" action="copy"/>
  <add as="Ctrl+G" action="group"/>
  <!-- Auto-wired by KeyHandler class -->
</keyhandler>
```

#### Toolbar Buttons (Auto-generated)
```xml
<toolbar>
  <add as="Save" action="save" icon="save.png"/>
  <add as="Undo" action="undo" icon="undo.png"/>
  <add as="Group" action="group" icon="group.png"/>
  <!-- Auto-generated by EditorToolbar class -->
</toolbar>
```

#### Cell Styles (Railway-specific)
```xml
<styles>
  <add name="station" style="shape=rectangle;rounded=1;..."/>
  <add name="signal-head" style="shape=rectangle;strokeColor=#D32F2F;..."/>
  <add name="track" style="shape=line;strokeColor=#8B4513;strokeWidth=3"/>
  <!-- Accessible in graph.getDefaultStylesheet() -->
</styles>
```

#### Shape Palette (Auto-loaded)
```xml
<shapePalette>
  <set name="Stations">
    <shape name="Major Station" style="station"/>
    <shape name="Platform" style="platform"/>
  </set>
  <set name="Signals">
    <shape name="Signal Head" style="signal-head"/>
    <shape name="Balise" style="balise"/>
  </set>
</shapePalette>
```

#### Grid & Snap (Auto-configured)
```xml
<grid enabled="true" size="40"/>
<snap enabled="true" size="10"/>
```

#### Editor Behavior (Auto-applied)
```xml
<edit>
  <autoSizeCells value="false"/>
  <cellsMovable value="true"/>
  <cellsResizable value="true"/>
  <cellsDeletable value="true"/>
</edit>
```

## Railway Drawer Specific Styles

Defined in config for railway domain:

**Track Elements**
- Main Track: thick brown line
- Parallel Track: parallel brown lines
- Platform Track: lighter track

**Stations**
- Major Station: blue rounded rectangle
- Minor Station: green rectangle
- Platform: gray rectangle
- Depot: yellow container

**Signaling (ERTMS)**
- Signal Head: red rectangle with border
- Stop Aspect: red circle
- Proceed Aspect: green circle
- Balise: purple circle (transponder)

**Junctions**
- Simple Junction: orange rectangle
- Diamond Junction: orange diamond
- Crossover: light orange rectangle
- Switches: blue rectangles

**Speed Management**
- Speed Limit: pink rectangle
- Speed Restriction: orange rectangle

**Connectors**
- Main Line: thick black orthogonal edge
- Branch Line: dashed gray edge
- Signal Link: red thick edge

## Extending Functionality

**To add a new feature:**

1. **Simple configuration change?**
   - Edit XML
   - Done!

2. **New cell style?**
   ```xml
   <add name="new-style" style="shape=rectangle;..."/>
   ```

3. **New keyboard shortcut?**
   ```xml
   <add as="Ctrl+L" action="custom"/>
   ```

4. **New action handler?**
   ```typescript
   editor.addAction('custom', () => { /* logic */ });
   ```

**That's it! No toolbar rebuilding, no event handler rewiring.**
